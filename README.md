# 智能对话Chatbot系统

基于 Python + Streamlit 框架开发的网页版智能对话系统，支持多轮对话、模型切换、对话历史管理等功能。

## 🎯 项目特点

- **模拟模式**：默认使用模拟输出，无需API密钥即可体验对话功能
- **扩展接口**：预留真实API接口，可快速接入OpenAI、Anthropic等模型
- **完整功能**：对话历史管理、导出功能、主题切换等完整功能

## 🚀 功能特性

### 💬 核心对话功能
- **实时对话**：支持用户文本输入与AI模型实时回复
- **多轮对话**：完整的上下文记忆，支持连续对话
- **流式输出**：实时显示AI响应，无需等待完整回复
- **状态反馈**：加载动画、思考提示、错误提示

### 📊 对话历史管理
- **会话分组**：自动管理多个对话会话
- **会话切换**：轻松在不同会话间切换
- **会话搜索**：支持关键词搜索历史对话
- **导出功能**：支持TXT和Markdown格式导出
- **一键复制**：快速复制对话内容

### ⚙️ 模型配置
- **多模型支持**：GPT-3.5、GPT-4、Claude等
- **API配置**：支持OpenAI、Anthropic等API
- **参数调节**：可调节temperature、max_tokens、top_p等参数
- **系统提示词**：自定义AI角色和行为

### 🎨 界面特色
- **双栏布局**：侧边栏配置+主对话区
- **消息气泡**：用户/助手消息样式区分
- **主题切换**：支持浅色/深色模式
- **响应式设计**：适配不同屏幕尺寸

## 📦 环境配置

### 系统要求
- Python 3.9+
- Windows/Linux/Mac

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/你的用户名/chatbot-project.git
cd chatbot-project
```

2. **创建虚拟环境（推荐）**
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **配置API密钥**
首次运行会提示输入API密钥，也可以在侧边栏设置中配置。

## 🎮 运行说明

### 启动应用
```bash
streamlit run app.py
```

应用将在浏览器中打开，默认地址：`http://localhost:8501`

### 基本使用

#### 模拟模式（默认）
1. 启动应用后默认运行在模拟模式
2. 无需配置API密钥即可体验对话功能
3. 支持基本的对话交互和历史管理
4. 在输入框输入问题开始对话

#### 真实API模式
1. 在侧边栏取消勾选"使用模拟模式"
2. 输入您的API密钥（OpenAI或Anthropic）
3. 选择要使用的AI模型
4. 调节参数（可选）
5. 在输入框输入问题开始对话
6. 对话会自动保存，可随时查看历史

## 🎯 主要模块

### config.py - 配置管理

- 模型参数管理
- 主题颜色配置
- API配置管理

### history_manager.py - 历史管理

### chat_utils.py - 工具函数

- 流式输出处理
- Token计数

### app.py - 主界面
- Streamlit界面渲染
- 侧边栏配置面板
- 对话消息展示
- 用户交互处理

## ⚙️ 参数说明

| 参数 | 说明 | 建议范围 |
|------|------|----------|
| Temperature | 控制输出随机性 | 0.5-0.9 |
| Max Tokens | 单次回复最大长度 | 1000-2000 |
| Top P | 核采样概率 | 0.9-1.0 |

## 🐛 常见问题

### 1. API请求失败
- 检查API密钥是否正确
- 确认API额度是否充足
- 检查网络连接

### 2. 响应速度慢
- 尝试降低max_tokens参数
- 选择响应更快的模型
- 检查网络延迟

### 3. 会话未保存
- 检查存储文件权限
- 确认磁盘空间充足
- 查看日志中的错误信息

## 🔧 开发说明

### 添加新模型支持
在`config.py`的`get_available_models()`函数中添加新模型配置。

### 自定义主题
修改`config.py`的`get_theme_colors()`函数中的颜色配置。

### 扩展导出格式
在`history_manager.py`的`export_conversation()`方法中添加新格式支持。
