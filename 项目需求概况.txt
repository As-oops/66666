# 项目名称：星期八 AI 对话助手（Streamlit Chatbot）
你现在需要开发一套网页版智能对话Chatbot系统，使用 Python+Streamlit 框架，适配 Windows 本地运行。

## 产品功能需求
1. 核心对话交互模块
- 支持用户文本输入与助手回复交互，实现多轮对话上下文记忆，支持模拟回复与真实模型流式输出
- 对话状态反馈（加载动画、思考提示、错误提示），输入防抖与消息长度限制

2. 对话历史管理模块
- 对话会话分组存储，支持新建/切换/删除会话，历史对话分页展示与关键词搜索
- 对话内容导出功能（TXT/Markdown格式），一键复制对话文本与重置会话功能
-上下文记忆：在代码里用列表存储历史对话，每一轮都把「历史对话 + 当前用户提问」一起传入模型，即可实现记忆功能。

3. 模型配置与参数调节模块
- 侧边栏配置面板，支持 DeepSeek 等多模型切换（当前以 DeepSeek 为主模型）
- 可调模型核心参数（temperature、max_tokens、top_p等），参数修改实时生效

4. 用户界面与体验优化模块
- 双栏布局（侧边栏配置/历史区 + 主对话区），对话气泡区分用户/助手消息，样式美观
- 支持自定义Streamlit主题（背景色、按钮配色、字体样式），适配深色/浅色模式
- 自定义助手形象与名称：助手固定名称为「星期八」，对话中以「星期八」身份进行回复

## 项目规范与质量要求
- 代码模块化封装（对话渲染函数、历史管理函数、模型调用函数、导出函数、界面配置函数），注释清晰
- 无报错、无警告，兼容本地稳定循环运行，适配 Python 3.9+ 版本
- 项目全程采用 Git 进行版本管理，提交规范清晰，最终推送至 GitHub 公开仓库
